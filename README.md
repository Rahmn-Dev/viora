## 🧊 About VIORA

**VIORA** is a modern movie streaming platform concept focused on immersive visuals, smooth interaction, and futuristic interface design.

Built using:

- ⚡ Vue.js for reactive frontend experience
- 🐍 Django for powerful backend architecture
- 🧊 Liquid Glass UI for premium modern aesthetics

VIORA combines cinematic presentation with sleek glassmorphism-inspired components to create a premium streaming experience.
